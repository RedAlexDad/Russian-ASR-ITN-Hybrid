# Словари и ASR-ошибки

## Структура словарей

Все словари расположены в `src/dicts/` и имеют единый формат:

```python
слово: (числовое_значение, порядок_величины)
```

### `units.py` — единицы, 10-19, десятки

| Раздел        | mag | Примеры                                     |
| ------------- | --- | ------------------------------------------- |
| Единицы 0-9   | 0   | `один: (1,0)`, `пять: (5,0)`                |
| Числа 10-19   | 1   | `одиннадцать: (11,1)`, `пятнадцать: (15,1)` |
| Десятки 20-90 | 2   | `двадцать: (20,2)`, `пятьдесят: (50,2)`     |
| Собирательные | 0   | `двое: (2,0)`, `трое: (3,0)`                |

### `hundreds.py` — сотни

| mag | Примеры                           |
| --- | --------------------------------- |
| 3   | `сто: (100,3)`, `двести: (200,3)` |

Включает ASR-искажения: `двеси: (200,3)`, `дваста: (200,3)`.

### `thousands.py` — тысячи, миллионы, миллиарды (умножители)

| Раздел    | mag | Примеры                             |
| --------- | --- | ----------------------------------- |
| Тысячи    | 4   | `тысяча: (1000,4)`, `тыщ: (1000,4)` |
| Миллионы  | 5   | `миллион: (1000000,5)`              |
| Миллиарды | 6   | `миллиард: (1000000000,6)`          |

Слова-умножители собраны в `MULTIPLIERS` — парсер использует их для
умножения накопленного значения.

### `ordinals.py` — порядковые числительные

Формат: `слово: "цифровое_значение"`

```python
'первый': '1', 'второго': '2', 'тринадцатое': '13'
```

## Функции поиска (`src/lexicon.py`)

### `lookup_word(word)`

Поиск слова в словаре:

1. Прямой поиск в `NUMERAL_DICT` (O(1))
2. Если не найдено — поиск в `ASR_ERRORS`
3. Возвращает `(значение, mag, is_multiplier)` или `None`

### `is_ordinal_word(word)`

Проверка, является ли слово порядковым числительным (O(1)).

### `ordinal_value(word)`

Возвращает цифровое значение порядкового (например, `"первого"` → `"1"`).

## ASR-ошибки (`src/dicts/asr_errors.py`)

Словарь частых искажений, собранный из анализа calibration.f:

```python
'двеси': 'двести',          # 44+ вхождения
'тыщ': 'тысяч',             # 14 вхождений
'петьсят': 'пятьдесят',
'пятнадцить': 'пятнадцать',
```
