# Гибридный подход: парсер + ruT5

Гибрид объединяет rule-based парсер (99.80%) с нейросетевым fallback (ruT5-small)
для обработки сложных случаев. Подробнее об архитектуре: [docs/architecture.md](architecture.md)

## Логика работы

```
Входной текст
    │
    ▼
┌───────────────────┐
│  Парсер           │ → pred_rule + confidence
│  (нормализатор)   │
└──────┬────────────┘
       │
       ▼
┌───────────────────┐
│  Оценка           │ → confidence ≥ 0.75 → pred_rule (ответ)
│  уверенности      │
│  parser_confidence│ → confidence < 0.75 → fallback на ruT5
└──────┬────────────┘
       │
       ▼
┌───────────────────┐
│  ruT5-small       │ → model.generate() → pred_t5
│  (LoRA)           │
└───────────────────┘
       │
       ▼
     Ответ
```

## Когда парсер сомневается

| Признак                              | Снижение confidence |
| ------------------------------------ | ------------------- |
| Найдены ASR-искажения (двеси, тыщ)   | -0.15 за каждое     |
| Слова вне словаря                    | -0.1 за каждое      |
| Склейки числительных (дветысячи)     | -0.2                |
| Были цифры в исходнике, нет в ответе | -0.3                |

Порог fallback: **confidence < 0.75**.

## Текущее состояние

| Режим               | Accuracy на calibration.f |
| ------------------- | ------------------------- |
| Парсер (current)    | 99.80%                    |
| Парсер (sequence)   | 99.80%                    |
| Гибрид (clean T5)   | 94.4% (хуже парсера)     |
| Гибрид (noisy T5)   | Не завершён               |

**Гибрид пока не используется** — чистый парсер даёт 99.80%, а T5,
обученный только на clean данных, снижает accuracy. Требуется
завершить обучение на noisy данных (см. [training.md](training.md)).

## Загрузка модели

Модель загружается **лениво** — только при первом fallback.
Если модели нет (`models/ruT5-itn/` не существует), гибрид работает
как обычный парсер без ошибок.

```python
from src.hybrid import hybrid_normalize
result = hybrid_normalize("бюджет двеси пятьдесят тысяч")
```

## Код

Модуль: `src/hybrid.py`
Функции:

- `hybrid_normalize(text)` — основная
- `parser_confidence(text, pred)` — оценка уверенности
- `_load_model()` — загрузка ruT5 + LoRA (lazy)
