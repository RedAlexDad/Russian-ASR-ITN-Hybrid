# План: Jupyter notebook для полного EDA

**Branch:** main
**Date:** 2026-06-05 18:31:34 MSK

## Мотивация

Текущий `make evaluate` выдаёт только текстовую статистику (accuracy, количество чисел, список ошибок).
Этого недостаточно для глубокого анализа данных. Требуется полноценный EDA в Jupyter с графиками,
гистограммами, heatmap и развёрнутыми markdown-комментариями.

## Данные

| Файл | Движок | Строк | Колонки |
|------|--------|-------|---------|
| `data/calibration.f` | Polars (IPC) | 500 | `task_text`, `ground_truth` |
| `data/test.f` | Polars (IPC) | 501 | `task_text` |

## Инструменты

- **Polars** — вместо Pandas (стандарт проекта)
- **Matplotlib + Seaborn** — визуализации
- **Jupyter nbconvert** — запуск и экспорт в HTML

## Файл

`notebooks/eda.ipynb` — самодостаточный ноутбук, читает данные из `data/`,
использует `src.normalizer.normalize_text` для оценки accuracy.

## Структура ноутбука (12 секций)

```
1. Подготовка окружения       — импорты, стиль графиков
2. Загрузка данных            — polars.read_ipc, schema, head
3. Первичный осмотр           — статистики, примеры, nulls
4. Распределение чисел        — гистограммы (кол-во на строку, разрядность)
5. Длина текста vs числа      — scatter + lmplot
6. Анализ ASR-ошибок          — частотность искажений (двеси, тыщ...)
7. Группировка (сумма vs перечисление) — ключевая сложность задачи
8. Accuracy решения           — оценка через normalizer, confusion
9. Детальный разбор ошибок    — классификация по типам, heatmap
10. Порядковые числительные   — отдельный анализ ordinals
11. Сравнение cal vs test     — распределения длин, доли чисел
12. Выводы и рекомендации     — итоги, следующие шаги
```

## Визуализации (8 графиков)

| # | График | Тип | Назначение |
|---|--------|-----|------------|
| 1 | Числа на строку | гистограмма | distribution чисел на фразу |
| 2 | Разрядность | гистограмма | 1-digit vs 2-digit vs ... |
| 3 | Строки с/без чисел | круговая | доля строк с числами |
| 4 | Длина vs Числа | scatter+lm | корреляция |
| 5 | Топ ASR-ошибок | barh | 20 частых искажений |
| 6 | Ошибки по типам | barh | classification ошибок |
| 7 | Путаница | heatmap | predicted vs actual |
| 8 | Cal vs Test | side-by-side | распределение длин |

## Makefile

```makefile
eda:       jupyter nbconvert --to html --execute notebooks/eda.ipynb
eda-view:  jupyter notebook notebooks/eda.ipynb
```

Цели добавляются в `makefiles/commands.mk` и `makefiles/help.mk`.
